
# LoopedIn: Find clubs to get involved with and discover vendors to collab with.

# Link to Devpost with video: https://devpost.com/software/loopedin 

LoopedIn connects students with student-run organizations on campus and allows organizations to connect with vendors for their events. Users can search, filter by tags, and "heart" their favorite clubs to pin them to their personal dashboard. 

LoopedIn is a web-based platform designed to connect students with student-run organizations on campus while also enabling those organizations to collaborate with vendors for events. The application allows users to browse clubs, filter them based on interests or “vibes,” and save their favorites using a heart feature that pins them to a personalized dashboard. This project was built using React for the frontend, along with HTML and CSS to design a clean and responsive interface. I focused on making the layout easy to navigate, with a simple two-panel design for discovering and saving clubs. One challenge I ran into was making sure the filtering and saved clubs updated correctly across the app, but I was able to fix that by organizing the components better and improving how the data was handled.
